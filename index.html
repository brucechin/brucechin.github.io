About Me

I graduated from Shanghai Jiao Tong University with a B.Eng. in Computer Software Engineering in 2019 summer. Now I am a first year Computer Science Ph.D student at Northwestern University. I am honored to be temporarily advised by Prof. Xiao Wang. I work closely with Prof. Shumo Chu.

My main research interest lies in applying modern cryptography techniques like secure computation, zero-knowledge proof and blockchain into data processing and storage system in untrusted environment. Although modern cryptography has made amazing progress over the past decades, few of these theoretical advances have been incorporated into practical systems. I believe there are opportunities via a co-design of systems and cryptography.

I used to work on an OLTP database system that supports Oblivious Transaction using Secure Multiparty Computation and Oblivious Parallel RAM for supporting secure outsourcing to cloud database infrastructure.
